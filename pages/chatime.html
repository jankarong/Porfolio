Introduction

Chatime's visually appealing website aims to captivate customers, but its usability issues hinder its full potential. To address these challenges, we focused on increasing app downloads, boosting promotional drink sales, and enhancing the overall user experience. By analyzing user behaviour and conducting thorough testing, we identified key areas for improvement. Our recommendations include optimizing promotional banners, streamlining the ordering process, and creating a more intuitive drink menu layout.

Hypotheses

Hypothesis 1

We believe that a 25% increase in drink orders will be achieved if new customers successfully identify consistent call-to-action buttons.

Hypothesis 2

We believe that a 30% increase in app downloads will occur if users can easily locate the app download section.

Hypothesis 3

We believe that 20% more promotional drinks will be sold if the promotional section is more prominently displayed.

Test Results

Order a drink for pickup

ASSUMPTION: USER FINDS ORDERING PROCESS OVERWHELMING

Observations:

  • Users had trouble finding the "Order Now" call to action.
  • Users scrolled to the footer, where the biggest call to action was located, spending more time than intended.
  • Everyone completed the task successfully with the longest time being 2.06 seconds, because of loading time.

Find promotional drinks

ASSUMPTION: PROMOTIONAL CONTENT IS HARD TO LOCATE

Observations:

  • Users struggled to find promotional content on the homepage.
  • The promotional banner was not prominent enough.
  • Users expected promotional content to be featured at the top of the page.

Download the mobile app

ASSUMPTION: APP DOWNLOAD SECTION IS NOT VISIBLE

Observations:

  • Users couldn't find the app download links easily.
  • The app benefits were not clearly communicated.
  • Mobile users expected a pop-up prompt for app download.

Customize drink options

ASSUMPTION: CUSTOMIZATION PROCESS IS CONFUSING

Observations:

  • Users found the customization options overwhelming.
  • The order of customization steps wasn't intuitive.
  • Users wanted clearer visual indicators for their selections.

SUS Calculation

SUS Score Analysis

Our System Usability Scale (SUS) evaluation revealed:

  • Overall SUS Score: 68%
  • High Score Range: 72.5%
  • Average Score Range: 61%
  • Low Score Range: 47.5%

Key Findings

  • The website scored slightly above the global average of 68%
  • Users found the basic functionality acceptable but indicated room for improvement
  • Navigation and layout issues were the main factors affecting the score

Recommendations

UI Elements

  • Redesign banner interaction.
  • Redesign and improve visibility of CTAs.
  • Optimize the 'drinks' page for desktop, utilizing card layout.
  • Replace hamburger menu with navigational bar for desktop.

Other

  • Optimize images to decrease loading time.
  • Connect 'Our Drink' page to the ordering page.
  • Dedicate a separate section for the App of the home page
